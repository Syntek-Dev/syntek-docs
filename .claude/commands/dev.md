Start the documentation site in local development mode with hot reload.

**Run:** `npm start`

This will:
1. Install dependencies if not already installed (`npm install`)
2. Start the Docusaurus dev server at http://localhost:3000
3. Hot-reload on file changes

**Useful flags:**
```bash
npm start -- --port 3001    # Use a different port
npm start -- --locale en    # Specific locale
```
